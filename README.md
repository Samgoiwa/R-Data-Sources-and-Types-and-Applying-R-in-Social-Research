 **Project Description: R Data Sources and Types and Applying R in Social Research**

This project explores the use of the R programming language for acquiring, managing, analyzing, and visualizing social research data. It begins with an overview of common data sources in social sciences—such as surveys, administrative records, and open data portals—and the data types typically encountered (categorical, continuous, ordinal, date-time).

Using a simulated social survey dataset, the project demonstrates how to:

* Import and clean data from various formats (CSV, Excel)
* Perform descriptive statistics and data transformations
* Apply visual and statistical methods to uncover patterns in civic participation, education, income, and trust in government
* Build predictive models using regression techniques
* Visualize correlation matrices to understand inter-variable relationships

By leveraging R packages like `tidyverse`, `ggplot2`, `caret`, `corrplot`, and `ggcorrplot`, this project showcases a practical and reproducible workflow for social science researchers. It emphasizes transparency, reproducibility, and actionable insights from data to inform public policy and social programs.
